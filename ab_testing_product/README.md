# Анализ A/B-теста нового алгоритма рекомендаций 

##  Бизнес-задача
Развлекательное приложение с «бесконечной» лентой контента монетизируется по двум моделям: платная подписка (без рекламы) и показ рекламы в ленте. Команда ML разработала новый рекомендательный алгоритм, чтобы показывать более релевантный контент и повысить вовлеченность пользователей (что напрямую влияет на просмотры рекламы и ценность подписки). 
**Задача:** проверить корректность проведения A/B-теста и статистически оценить влияние нового алгоритма на метрики активности.

##  Инструменты и стек
* **Язык:** Python
* **Библиотеки:** `pandas`, `matplotlib`, `scipy.stats`, `statsmodels`

##  Краткое описание этапов работы
1. **Исследовательский анализ данных:** 
   - Загружены логи сессий пользователей.
   - Проверена система сплитования: выявлены и изолированы пользователи, ошибочно попавшие одновременно в контрольную (старый алгоритм) и тестовую (новый алгоритм) группы.
2. **Формирование метрик:** 
   - Рассчитана конверсия в целевое действие («успешная» сессия с высокой глубиной просмотра ленты контента).
   - Подсчитана базовая конверсия и средняя активность для обеих групп.
3. **Статистическая проверка гипотез:**
   - Для сравнения конверсий (долей успешных сессий) применен Z-тест для пропорций (`proportions_ztest`).
   - Для сравнения средних значений пользовательской активности применен T-тест Стьюдента (`ttest_ind`).

##  Главный вывод
*Статистические тесты (p-value < 0.05) подтвердили успешность нового алгоритма. Доля сессий с высокой глубиной просмотра в тестовой группе значимо превысила показатели контрольной группы (31.8% против 30.7%). Рекомендуется внедрить новый алгоритм рекомендаций на всю аудиторию, так как рост вовлеченности напрямую увеличит рекламную выручку.*
